This is a Live Now Widget for the Talkshoe.com users to be able to tweet to their own Twitter accounts and launch the Live Pro Client.

== Description == 

This plugin is a widget that allows Talkshoe users to easily tweet to their own Twitter accounts and launch the Talkshoe Live Pro Client. It uses some of the existing Talkshoe API code along with PHP to achieve a decent way to give your visitors more options for promoting Talkshoe calls.
 
To use this plugin once installed and activated, simply go to your Widgets page in your Admin area and you will find the WP Talkshoe Live widget there, drag it to your sidebar and you are finished!  No options to set in this widget!

= When a call is live, the widget will display: =
* The Show Name and Call ID with a clickable link to the show page on Talkshoe.
* A Tweet Now button.

* The Show Name and Call ID with a clickable link to the show page on Talkshoe.
* A Tweet Now button
* A button to launch the Talkshoe Live Pro Client
* A message that the call is not currently streaming

This plugin has been tested on version 3.0 of Wordpress, but may work on previous versions too.

== Installation ==

1. Upload `wptalkshoelive.php` to the `/wp-content/plugins/` directory or install via New Plugins section.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the widget called WP Talkshoe Live and drag it to your sidebar.
4. To use in your templates, posts or pages, etc. use '[wp_talkshoe_live]' (without the quotes).
